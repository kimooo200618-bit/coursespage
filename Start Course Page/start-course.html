About this course

You interact with JavaScript code all the time — you just might not realize it. It powers dynamic behavior on websites and plays an important role in many fields, like front-end and back-end engineering, game and mobile development, virtual reality, and more.

In this course, you'll learn JavaScript fundamentals that will be helpful as you build websites, web applications, and more. Starting with the basics like variables, data types, and functions, you'll gradually work your way up to more advanced concepts like object-oriented programming and asynchronous JavaScript.

Skills you'll gain

  • Build core programming concepts
  • Learn object-oriented concepts
  • Read and write JavaScript
  • Work with arrays and objects
  • Handle asynchronous code with Promises
  • Manipulate the DOM for web interactivity
